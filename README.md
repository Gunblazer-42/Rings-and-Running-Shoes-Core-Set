# Rings & Running Shoes Core Set
A FoundryVTT version of the core rulebook for Rings &amp; Running Shoes, a Powered by the Apocalypse-based, Sonic the Hedgehog-themed tabletop role playing game. This isn't required to run the system in Foundry, but it, along with the PBTA configuration sheet included in this repository, helps automate the process of managing the playbooks and rolls.
